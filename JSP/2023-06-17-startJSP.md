JSP(+서블릿) ---> 웹사이트 개발(+웹서버: Apache Tomcat)

JSP(Java Server Page): 자바코드 + html  프론트엔드까지 다루도록, 동적 웹페이지를 설정하기 위해 --->톰캣(서블릿 컨테이너)
jsp 코드 -> 서블릿 -> 톰캣(적재) -> 브라우저(출력)
