<h1>제어문(조건문과 반복문)</h1>

<h3>조건문</h3>

**조건연산자** (크기 비교나 동일한지의 여부를 체크함)<br>
(1) 크기 비교(> < >= <=)
(2) 동일한지의 여부 (== !=)
<br><br>
**변수 2개에 데이터를 저장하고 크기 비교하기**<br>
(1) data1 = 1;  data2 = 2;
(2) 크기 비교하기 (data > data2)
(3) boolean check;
(4) check = (data1 > data2);
<br><br>
**자바에서는 문자열 비교시 ==나 !=를 사용하면 안됨**<br>
